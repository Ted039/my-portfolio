Project 1 Details

This project is a responsive web application that allows users to track their daily tasks. Built using HTML, CSS, and JavaScript, it enables users to add, update, and remove tasks dynamically.

Technologies used: HTML, CSS, JavaScript, and Bootstrap for responsive design.

Key Features:

  • User authentication (login and signup)
  • Task filtering by priority
  • Responsive design for mobile and desktop
  • Real-time task updates using local storage

Challenges faced included managing asynchronous requests and ensuring data was saved to local storage. I overcame this by learning about the Fetch API and using JavaScript promises.

Future improvements: I plan to implement user authentication with Firebase and add a feature for recurring tasks.

Project 2 Details

This project is a personal portfolio website built using HTML, CSS, and JavaScript. It showcases various web development projects and provides an interactive user experience.

Technologies used: HTML, CSS, JavaScript, and a bit of jQuery for smooth animations.

Key Features:

  • Interactive portfolio with smooth transitions
  • Responsive design for both desktop and mobile
