Community Harvest Initiative

This project is a web application designed to help community members track, manage, and share agricultural resources. Built using Flask, MySQL, HTML, Bootstrap, and CSS, it allows users to register, log in, and update information about available resources and needs within the community.

Technologies used: Flask for the back-end, MySQL for the database, and HTML, Bootstrap, and CSS for the front-end design.

Key Features:

  • User registration and authentication
  • Add and update community resources such as food, tools, and seeds
  • Search functionality for available resources
  • Responsive design for both mobile and desktop users
  • Admin panel for managing user data and resources

Challenges faced included managing user authentication securely and ensuring proper data handling with MySQL. I overcame these challenges by implementing Flask's built-in security features and learning how to properly manage database queries with SQLAlchemy.

Future improvements: I plan to integrate real-time chat functionality for community members to communicate and improve the system's scalability to handle larger data sets as the community grows.
